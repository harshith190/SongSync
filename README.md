SongSync 🎵

An AI-powered music discovery web app that identifies songs playing in videos, reels, and shorts, then instantly opens them on Spotify. Built with a modern React frontend and responsive UI, SongSync helps users discover, save, and enjoy music seamlessly across social media platforms.

##LIVE DEMO [https://drive.google.com/file/d/1xHvD6yLxFVA8E-An0m0tWQHF8s6XEBsL/view?usp=drivesdk](url)

Tech Stack: React, JavaScript, Spotify API, REST APIs, HTML, CSS

Key Features:

Song recognition from video/audio clips
Direct Spotify integration
Modern Netflix-inspired UI
Responsive design for mobile and desktop
Fast and user-friendly music discovery experience
